A modern, desktop-based productivity tool designed to manage daily and weekly tasks efficiently. Built with C++ and Qt Framework, using SQLite for persistent data storage.

## ✨ Features
* **Daily & Weekly Management:** Separate views for tomorrow's tasks and the entire week.
* **Smart Progress Tracking:** Independent progress bars for daily and weekly goals.
* **Persistent Storage:** Integrated SQLite database to keep your tasks safe between sessions.
* **Data Export:** Export your task summaries to `.txt` reports with one click.
* **Modern UI:** Dark mode interface with customized Qt Style Sheets (QSS).

## 🛠 Tech Stack
* **Language:** C++ (C++17 or newer)
* **Framework:** Qt 6.x (Widgets module)
* **Database:** SQLite
* **Styling:** Custom QSS (Dark Theme)

## 🚀 Installation & Build
1. Clone the repository: `git clone https://github.com/YourUser/FocusFlow.git`
2. Open the project in **Qt Creator**.
3. Choose the **Desktop Qt 6.x MinGW** kit.
4. Build and Run!
